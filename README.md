# 终端命令行 CPU 烤机工具 (CPU Stress Test)

[![License](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)
![Platform](https://img.shields.io/badge/Platform-Linux%20%7C%20macOS-lightgrey)
![Language](https://img.shields.io/badge/Language-Bash-green)

一款轻量级的纯 Shell 脚本 CPU 压力测试工具，用于在终端中快速拉满 CPU 负载，检测系统稳定性、散热性能及硬件极限。

## 📖 简介

本项目提供了一个简单的 Shell 脚本 (`cpu-test.sh`)，通过创建大量后台进程执行高强度数学运算（大整数乘法、浮点数除法及字符串处理），从而将 CPU 占用率提升至接近 100%。脚本内置了从 1 到 512 线程的可视化菜单，方便用户根据自己的配置选择测试等级。

> **注意**：本工具主要用于学习操作系统进程管理、Shell 编程以及硬件压力测试。

## ✨ 功能特性

*   **纯 Shell 实现**：无任何外部依赖（仅需 `bash`/`sh`），兼容性强。
*   **可视化菜单**：提供 1-512 线程的快捷选择菜单。
*   **动态负载**：通过多层循环嵌套确保 CPU 处于高负荷状态。
*   **跨平台**：支持Android、Linux、macOS（Intel/Apple Silicon）。

## ⚠️ 风险提示（必读）

**在使用本工具前，请务必阅读以下内容：**

1.  **硬件风险**：长时间满负荷运行会导致 CPU 温度急剧升高，可能触发系统过热保护（降频或关机），在极端散热不良的情况下存在**缩短硬件寿命或损坏硬件**的风险。
2.  **系统稳定性**：脚本会创建大量进程，可能导致系统响应迟缓、卡顿甚至假死。请确保保存好重要工作后再运行。
3.  **服务器/云环境禁用**：
    *   **切勿在共享服务器、生产环境或云服务器（VPS）上运行此脚本**，除非你清楚知晓服务商条款。
    *   云厂商通常会将此类行为判定为“滥用计算资源”或“挖矿行为”，可能导致**账户封禁、IP封锁或罚款**。
4.  **强制重启**：如果系统完全卡死，唯一的退出方式是长按电源键强制重启，这可能导致未保存的数据丢失。

**本人不对因使用本工具造成的任何硬件损坏、数据丢失或服务封禁承担任何责任。**

## 🚀 使用方法

### 1. 下载脚本
```bash
git clone https://github.com/gitlune/CPU-TEST.git
```
### 2.进入文件夹
```bash
cd CPU-TEST
```
### 3. 赋予执行权限
```bash
chmod +x cpu-test.sh
```
### 4. 运行脚本
```bash
bash cpu-test.sh
```
### 5. 然后自行选择线程数即可
